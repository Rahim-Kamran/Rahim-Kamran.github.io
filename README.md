# David Chu's China Bistro 🍜

A responsive restaurant website built as part of the Coursera 
Full-Stack Web Development course (Module 5 Assignment).

## 🌐 Live Demo
[View on GitHub Pages](https://rahim-kamran.github.io/module5_solution/)

## 📌 Features
- Dynamic menu loaded via AJAX
- Responsive layout using Bootstrap
- Mobile-friendly collapsible navigation
- Google Fonts typography (Oxygen & Lora)
- Kosher certification display
- Click-to-call button on mobile
- Delivery info and business hours in footer

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 3
- jQuery
- Vanilla JavaScript (AJAX)

## 🚀 Getting Started
1. Clone the repository
   git clone https://github.com/Rahim-Kamran/Rahim-Kamran.github.io.git
2. Navigate to the project folder
   cd module5_solution
3. Open index.html in your browser
